GRC Hardening Compliance Audit (CIS Aligned)
Overview

This project demonstrates a Windows and Linux hardening compliance assessment aligned with CIS Benchmarks.

Scope
Windows 11 security configuration
Kali Linux hardening validation
Firewall configuration (Windows Defender & UFW)
SSH access control enforcement
Key Findings
Weak password policy identified (minimum length = 0)
Firewall required validation across profiles
SSH root login required hardening
Remediation Actions
Enforced password policy improvements
Verified firewall activation across all profiles
Disabled SSH root login (PermitRootLogin no)
Enabled UFW firewall in Linux
Outcome
Improved control effectiveness from ~50% to 90%+
Reduced critical risks through remediation
Aligned system with CIS security benchmarks
